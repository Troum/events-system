# 📐 Grid Layout для BookingForm

## Что изменилось

Форма бронирования теперь использует CSS Grid для оптимального размещения полей и занимает всё доступное пространство.

## Основные изменения

### 1. Grid Layout для полей

**До:**
```vue
<div class="space-y-5">
  <UFormField label="Ваше имя" />
  <div class="grid grid-cols-1 md:grid-cols-2 gap-5">
    <UFormField label="Телефон" />
    <UFormField label="Email" />
  </div>
</div>
```

**После:**
```vue
<div class="grid grid-cols-1 lg:grid-cols-2 gap-6">
  <UFormField label="Ваше имя" class="lg:col-span-2" />
  <UFormField label="Телефон" />
  <UFormField label="Email" />
</div>
```

### 2. Увеличенные размеры

| Элемент | Было | Стало |
|---------|------|-------|
| Заголовки секций | `text-lg` | `text-xl font-bold` |
| Иконки в заголовках | `w-5 h-5` | `w-6 h-6` в контейнере `w-10 h-10` |
| Отступы между секциями | `space-y-5` | `space-y-6` |
| Отступы между полями | `gap-5` | `gap-6` |
| Закругления полей | `rounded-xl` | `rounded-2xl` |
| Padding полей | `px-5 py-4` | `px-6 py-5` |
| Размер текста в полях | `text-lg` | `text-xl` |
| Размер счетчика мест | `text-2xl` | `text-3xl py-6` |
| Итоговая сумма | `text-3xl` | `text-4xl` |
| Кнопка отправки | `text-lg py-5` | `text-xl py-6` |
| Иконка кнопки | `w-6 h-6` | `w-7 h-7` |

### 3. Улучшенные заголовки секций

**До:**
```vue
<h3 class="text-lg font-semibold flex items-center gap-2">
  <UIcon name="i-heroicons-user-circle" class="w-5 h-5 text-primary-500" />
  Контактные данные
</h3>
```

**После:**
```vue
<h3 class="text-xl font-bold flex items-center gap-3">
  <div class="w-10 h-10 rounded-xl bg-primary-100 dark:bg-primary-900/30 flex items-center justify-center">
    <UIcon name="i-heroicons-user-circle" class="w-6 h-6 text-primary-600 dark:text-primary-400" />
  </div>
  Контактные данные
</h3>
```

### 4. Grid структура

```
┌─────────────────────────────────────────────────────────┐
│  [Имя                                                 ] │  ← col-span-2
├──────────────────────────────┬──────────────────────────┤
│  [Телефон                  ] │  [Email                ] │  ← 2 колонки
└──────────────────────────────┴──────────────────────────┘
```

**Адаптивность:**
- **Mobile (< 1024px):** 1 колонка, все поля на всю ширину
- **Desktop (≥ 1024px):** 2 колонки, имя занимает обе

## CSS Grid преимущества

✅ **Гибкость** - легко менять количество колонок
✅ **Адаптивность** - автоматическая перестройка на мобильных
✅ **Контроль** - точное управление через `col-span`
✅ **Консистентность** - одинаковые отступы между всеми элементами
✅ **Простота** - меньше вложенных div'ов

## Адаптивные breakpoints

```css
/* Mobile first */
grid-cols-1          /* < 1024px: 1 колонка */

/* Desktop */
lg:grid-cols-2       /* ≥ 1024px: 2 колонки */
lg:col-span-2        /* Имя занимает обе колонки */
```

## Размеры компонентов

### Поля ввода (UInput)
```vue
:ui="{ 
  rounded: 'rounded-2xl',      // Закругления
  padding: { xl: 'px-6 py-5' }, // Отступы
  size: { xl: 'text-xl' }       // Размер текста
}"
```

### Счетчик (UInputNumber)
```vue
:ui="{ 
  base: 'rounded-2xl text-3xl font-bold text-center py-6'
}"
:increment="{ size: 'xl', class: 'rounded-2xl' }"
:decrement="{ size: 'xl', class: 'rounded-2xl' }"
```

### Блок итоговой стоимости
```vue
<div class="rounded-3xl p-8 shadow-lg">
  <span class="text-lg">...</span>        <!-- Метки -->
  <span class="text-xl">...</span>        <!-- Значения -->
  <span class="text-2xl">Итого</span>     <!-- Заголовок итого -->
  <span class="text-4xl">30 000 ₽</span>  <!-- Итоговая сумма -->
</div>
```

### Кнопка отправки
```vue
<UButton 
  class="text-xl py-6 rounded-2xl"
>
  <UIcon class="w-7 h-7" />
</UButton>
```
